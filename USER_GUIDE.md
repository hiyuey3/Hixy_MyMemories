# MyMemories 使用指南（精简版）

MyMemories 用于记录生活中的纪念日和倒数日。下面是快速上手与常用操作说明。

## 能做什么
- 添加/编辑/删除记忆事件（生日、纪念日、考试、旅行等）
- 显示“还有 X 天 / 已经 X 天”，倒数与纪念颜色区分
- 批量管理与删除确认（带震动反馈）

## 布局
- 底部标签导航：主页 / 记忆 / 关于
- 主页：统计卡片 + 列表 + “新增事件”入口
- 管理页：编辑、删除、批量操作
- 关于页：版本、开发团队与联系渠道

## 快速上手
1. 打开 DevEco Studio，导入项目
2. 选择设备并运行，进入首页
3. 点击“新增”添加事件，或进入“管理”进行批量操作

## 功能使用
- 首页：查看统计与列表，点击卡片查看详情
- 记忆（MemoriesPage）：用月历选择日期过滤，支持跳转到指定日期
- 详情页：编辑或删除当前事件
- 编辑页：填写标题/日期/类型/备注并保存
- 管理页（开发者模式）：导入/导出 JSON、清理和调试新增

## 常见问题
- 新增后未显示：稍后刷新或返回首页触发重载
- 日期显示异常：确认选择的日期格式并重新保存
- 删除无反应：检查是否点击了确认删除按钮
