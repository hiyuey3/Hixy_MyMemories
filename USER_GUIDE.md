# MyMemories 使用指南（精简版）

MyMemories 用于记录生活中的纪念日和倒数日。下面是快速上手与常用操作说明。

## 能做什么
- 添加/编辑/删除记忆事件（生日、纪念日、考试、旅行等）
- 显示“还有 X 天 / 已经 X 天”，倒数与纪念颜色区分
- 批量管理与删除确认（带震动反馈）

## 布局
- 底部标签导航：主页 / 记忆 / 关于
- 主页：统计卡片 + 列表 + “新增事件”入口
- 管理页：编辑、删除、批量操作
- 关于页：版本、开发团队与联系渠道

## 快速开始
1) 打开应用进入首页
2) 点击“新增事件”
3) 输入标题，选择日期与类型（倒数/纪念）
4) 保存后自动回到列表

## 管理事件
- 在“记忆”或“管理”页查看全部记录
- 点击进入详情，或在管理页批量编辑/删除
- 删除会弹出确认并提供震动反馈

## 使用小贴士
- 为重要日期设置成“倒数日”，更直观看到剩余天数
- 过去的特殊日子使用“纪念日”，可以查看累计天数
- 适当清理不再需要的记录，保持列表清爽

## 常见问题
- 新增后未显示：稍候刷新或重新进入页面；检查是否保存成功
- 日期显示异常：检查选择的日期格式；重新选择后保存
- 删除无反应：确认是否已点击确认；查看是否有震动反馈
