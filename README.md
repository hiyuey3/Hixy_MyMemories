Hixy_MyMemories
================

关于本应用
---------
Hixy_MyMemories 是一款简单轻量的“记忆/纪念日”管理应用，帮助你记录并提醒重要的日子：生日、纪念日、旅行、倒数日等。界面简洁，支持添加事件、查看倒数与回顾。

主要功能
-------
- 添加事件：记录标题、日期与类型（倒数日 / 纪念日）。
- 列表展示：按日期排序显示所有事件，区分已过和即将到来。
- 统计卡片：显示即将到来的事件数量和纪念回顾数量。
- 顶部合照位：页面顶部预留圆形合照位置，默认显示应用图片，支持自定义（可扩展）。
- 底部导航：页面底部有导航栏占位，可扩展为多个页面入口。

如何使用（简要）
---------------
- 打开应用：进入主页面可看到顶部合照、统计卡片与事件列表。
- 新增事件：点击 "+ 新增事件"，填写标题、选择日期与类型，点击保存。
- 编辑/删除：当前版本为基础Demo，未来可扩展事件的编辑与删除功能。
- 更换合照：合照位置为占位，后续可支持点击替换为自己的照片（请关注后续更新）。

隐私与数据
-----------
- 本应用示例默认将数据保存在本地（演示用途）。生产环境请根据需求添加安全存储或云端同步，并在隐私政策中声明数据使用方式。

常见问题
-------
Q: 我可以更改顶部合照高度或样式吗？  
A: 可以，应用界面文件中有配置项用于调整合照区域的高度与图片尺寸（需要开发工具修改并重新打包）。

Q: 底部导航如何跳转到其它页面？  
A: 当前为占位实现，若要支持页面跳转需在导航按钮中接入路由或页面切换逻辑。
